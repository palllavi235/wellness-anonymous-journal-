# 🌿 Wellness Center IIT Roorkee – Anonymous Mental Wellness Platform

## 📌 Overview

The **Wellness Center IIT Roorkee** platform is a web-based application designed to provide students with a safe, private, and judgment-free space to express their thoughts and emotions anonymously. The platform encourages self-reflection, emotional well-being, and mental health awareness by allowing users to maintain personal journal entries while preserving anonymity.

This project was developed to promote mental wellness within the IIT Roorkee community and create an accessible digital support space for students.

---

## ✨ Features

### 🔒 Anonymous Access

* Users can enter the platform using an anonymous username.
* No personal information is required.
* Focused on privacy and psychological safety.

### 📖 Personal Journaling

* Create anonymous journal entries.
* Record daily thoughts, feelings, and experiences.
* Encourage self-awareness and emotional reflection.

### 📝 Journal Management

* View previously written journal entries.
* Edit existing entries.
* Delete entries whenever desired.
* Maintain a personalized record of emotional growth.

### 🌱 Safe Wellness Space

* Non-judgmental environment.
* Encourages healthy emotional expression.
* Supports mental well-being and self-care practices.

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/wellness-center-iitr.git
```

2. Navigate to the project directory:

```bash
cd wellness-center-iitr
```

3. Open `index.html` in your browser.

No additional dependencies or installation steps are required.
